# mern-crud-auth

This project contains two parts: Backend and Client

The modules used in the backend are:

- Express
- mongoose
- morgan
- zod
- jsonwebtoken
- cors
- cookie-parser
- bcrypt
- dotenv

The modules used in the client are:

- axios
- react-hokk-form
- react-icos
- react-router-dom
- styled-components
